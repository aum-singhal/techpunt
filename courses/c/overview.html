History of C

C is a programming language that was initially developed by Dennis Ritchie as a system programming language to write an operating system.

Main features of C language:

  • low-level access to memory
  • simple set of keywords
  • clean style
  • Easy to learn
  • It produces efficient programs.
  • Structured language
  • It can handle low-level activities.
  • It is often compiled on a spread of computer platforms.

Which makes the C language suitable for system programmings like operating systems or compiler development.
