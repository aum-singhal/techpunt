Constants and Litterals

The constants ask fixed values that the program might not alter during its execution. These fixed values also are called literals. Constants are often of any of the essential data types like an integer constant, a floating constant, a personality constant, or a string literal. There also are enumeration constants also. The constants are treated a bit like regular variables except that their values can't be modified after their definition.

Integer literals

An integer literal is often a decimal, octal, or hexadecimal constant. A prefix specifies the bottom or radix: 0x or 0X for hexadecimal, 0 for octal, and zip for decimal. An integer literal also can have a suffix that's a mixture of U and L, for unsigned and long, respectively. The suffix can be uppercase or lowercase and can be in any order. Here are some examples of integer literals:

212 /* Legal */ 215u /* Legal */ 0xFeeL /* Legal */ 078 /* Illegal: 8 is not an octal digit */ 032UU /* Illegal: cannot repeat a suffix */

Following are other examples of various types of Integer literals:

85 /* decimal */ 0213 /* octal */ 0x4b /* hexadecimal */ 30 /* int */ 30u /* unsigned int */ 30l /* long */ 30ul /* unsigned long *

Floating-point literals

A floating-point literal has an integer part, a percentage point, a fractional part, and a lover part. you'll represent floating point literals either in decimal form or exponential form. While representing using the decimal form, you want to include the percentage point, the exponent, or both and while representing using the exponential form, you want to include the integer part, the fractional part, or both. The signed exponent is introduced by e or E.

Here are some samples of floating-point literals:

3.14159 /* Leagal */
314524E-5L /* Leagal */
510E /* Illegal Incomplete exponent */
210f /* Illegal no decimal or exponent */
-55 /* Illegal missing integer or fraction

String literals

String literals or constants are enclosed in double-quotes "". A string contains characters that are similar to character literals: plain characters, escape sequences, and universal characters. You can break a long line into multiple lines using string literals and separating them using whitespaces. Here are some examples of string literals. All three forms are identical strings.

"hello, dear" "hello, \ dear" "hello, " "d" "ear"

Defining Constants

There are two simple ways in C to define consatants:

  1. Using #define preprocessor
  2. Using const keyword

The #define preprocessor

Following is the form to use #define preprocessor to define a constant:

#include #define LENGTH 10 #define WIDTH 5 #define NEWLINE '\n' int main() { int area; area = LENGTH + WIDTH printf("value of the area : %d", area); printf("%c", NEWLINE); return 0; }

When the above code is compiled and executed, it will produce the following result:

value of area: 50

The const keyword

You can use the const prefix to declare constants with a specific type as follows:

const type variable = value;

The following example explains it in detail:

#include int main() { const int LENGTH 10 const int WIDTH 5 const char NEWLINE '\n' int area; area = LENGTH + WIDTH printf("value of the area : %d", area); printf("%c", NEWLINE); return 0; }

When the above code is compiled and executed, it produces the following result:

value of the area: 50

Note that it is a good programming practice to define constants in CAPITALS.
